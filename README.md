<h2>📋 Sobre o projeto</h2>
<p>Este é um projeto de um carrossel interativo e responsivo focado em uma experiência visual premium para a exibição de carros de luxo. O projeto utiliza transições suaves e 
  um design moderno com fundo em degradê radial para destacar os veículos.</p>

<p><h3>✦ Principais funcionalidades:</h3></p>

<p>➥ Navegação Fluida: Troca de slides através de setas laterais com suporte a loop infinito.</p>
<p>➥ Animações Sincronizadas: Os elementos de texto (título, descrição, botões) e a imagem do carro possuem tempos de entrada diferentes (delays) para criar um efeito visual dinâmico</p>
<p>➥ Indicadores de Progresso: <br>
    • Contador numérico dinâmico (ex: 01, 02, 03). <br>
    • Barras de progresso visuais que indicam em qual slide o usuário está.</p>
<p>➥ Efeito Parallax: As imagens dos carros rotacionam e se movem lateralmente durante a transição.</p>
